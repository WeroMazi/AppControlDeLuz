# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-05-05
- Added startup onboarding flow and improved typography/theme consistency.
- Upgraded room controls with brightness slider and visual level indicator.
- Added accessibility improvements (content descriptions and localized strings ES/EN).
- Added local cache fallback for light states and brightness values.
- Added Firebase Analytics and Crashlytics integration hooks.
- Added privacy notice dialog shown on first launch.
- Added unit tests for `AppViewModel` behavior and Compose UI test for dashboard.
- Added GitHub Actions CI workflow for lint, tests, assemble, and bundle.
- Hardened release build configuration and ProGuard rules for production.
